📝 CSS Hover Concepts (Handout Summary)
This project implements several concepts from the "CSS Hover Effects" handout:
Pseudo-class: :hover used for state changes.
Transition: cubic-bezier for non-linear, organic movement.
Transform: scale() and translateY() for depth.
Clip-path: Animating a circular reveal from the corner of the card.
